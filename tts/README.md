# TruthTrace

## **1. 프로젝트 개요**

- **프로젝트명:** **TruthTrace – AI 기반 정보 신뢰도 분석 플랫폼**
- **목적:** 웹상에 존재하는 기사·게시글·SNS 콘텐츠의 **신뢰도·편향성·근거 수준**을 AI가 자동 분석하여, 사용자가 보다 **객관적이고 비판적인 시각으로 정보를 소비**할 수 있도록 지원
- **형태:** 웹 기반 분석 도구 (MVP 단계에서는 텍스트 입력 → 분석 → 시각화 중심 기능 구현)
- **대상 데이터:** 뉴스 기사 본문, 블로그 게시글, SNS 텍스트(트윗, 커뮤니티 등)

---

## **2. 문제정의**

- 정보 과잉 시대에 사용자는 수많은 뉴스와 게시글 중 어떤 것이 **신뢰할 만한지 판단하기 어려움**.
- 가짜뉴스, 편향된 콘텐츠, 감정적 표현이 많아 **객관적 사실과 의견의 구분이 불명확**함.
- 현재의 뉴스 포털이나 SNS 플랫폼은 “정보를 제공”할 뿐, **신뢰도를 평가하거나 근거를 설명하는 기능이 부재**함.
- 따라서 **AI를 통한 자동 신뢰도 분석 및 피드백 제공**을 통해 정보 소비의 질을 높일 필요가 있음.

---

## **3. 핵심 타겟**

- **일반 사용자:** 뉴스, 블로그, SNS에서 접한 정보를 빠르게 검증하고 싶은 사람
- **콘텐츠 제작자:** 인용 정보의 신뢰성을 확인하고, 잘못된 정보를 걸러내고 싶은 크리에이터
    - **연구자/기자/정책 담당자:** 대량의 정보를 분석해 **신뢰도 높은 자료만 선별**하고 싶은 전문가

---

## **4. 핵심 컨셉**

> “AI가 읽고 판단하는 정보 신뢰도 분석 도구”
> 
> - 뉴스·SNS·게시글 입력 → **AI 신뢰도 평가 + 편향 감지 + 근거 문장 추출**
> - **시각적 점수화 및 피드백 제공**을 통해 한눈에 정보 신뢰 수준을 확인
> - 사용자는 단순한 ‘정보 소비자’에서 ‘비판적 판단자’로 전환

## **6. 차별성 (경쟁력 및 가치)**

1. **정량적 신뢰도 지표 제공**
    - 감정적 직관 대신 **AI가 수치화된 신뢰도 점수**를 제공하여 객관성 강화.
2. **편향성 감지 기능 내장**
    - 단순 사실 검증을 넘어 **언어적·이념적 편향**을 자동 감지.
3. **근거 기반 시각화**
    - 주장과 근거 문장을 연결해 **이해 가능한 신뢰도 구조 시각화** 제공.
4. **경량 AI 기술로 구현 용이**
    - GPT API 또는 DistilBERT 기반 감정/편향 분석으로 **MVP 수준에서도 충분히 구현 가능.**
5. **확장성 높은 구조**
    - 뉴스뿐 아니라 블로그, 커뮤니티, SNS 댓글 등 다양한 데이터 형태에 확장 가능.

---

## **7. 기대 효과**

- **사용자 측면:**
    - AI 피드백을 통해 정보의 진위 여부를 빠르게 판단하고, **비판적 사고력 향상**
    - 허위 정보 및 과장된 주장에 대한 **판단 피로 감소**
- **학습/교육 측면:**
    - NLP, 감정 분석, LLM 기반 신뢰도 예측 및 시각화까지 **AI 파이프라인의 전체 흐름 학습 가능**
    - 대학·연구 프로젝트에서 **사회적 가치와 기술적 성취를 동시에 실현 가능**
- **시장/서비스 측면:**
    - 가짜뉴스 검증, 브랜드 모니터링, 정책 여론 분석 등 **B2B 확장 가능성**
    - 신뢰도 분석 기능을 통해 **언론 및 공공기관 데이터 검증 솔루션**으로 발전 가능
- **기술 확장성:**
    - 크롬 확장 프로그램, 뉴스 브리핑 서비스 등으로 **실시간 웹 분석 도구로 발전 가능**
